# 2. 开始使用 Nuxt3

环境要求

如果你已经安装了 Node.js，请检查版本大于 16.11，命令：

node --version

推荐 VS Code 安装插件 TypeScript Vue Plugin (Volar)：

code --install-extension Vue.volar

首先创建项目：

npx nuxi init <project-name>

pnpm dlx nuxi init <project-name>

然后安装依赖：

cd <project-name>
yarn install

cd <project-name>
npm install

cd <project-name>
pnpm install

接着在 VS Code 中打开：

code .

启动开发模式：

yarn dev -o

npm run dev -- -o

pnpm dev -o

其中 -o 表示自动打开浏览器页面（http://localhost:3000在新窗口打开），这样你可以直接看到效果了，请等待一会完成初始渲染。
